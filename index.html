A propos d'Open OS...

Open OS est un systeme d'exploitation (en cours de creation) disponible sur toutes les platformes ! Il vous suffit d'un navigateur récent (Désolé Internet Explorer...) pour le faire fonctionner. Le système fonctionne sous JavaScript, HTML et CSS qui sont des languages servant a creer et faire fonctionner des sites web comme celui-ci !

Comment l'utiliser ?

Utiliser Open OS, c'est on ne peut plus simple !

1.

Téléchargez le launcher Open OS ansi qu'Open OS en js
(↑ C'est tout en haut !).

2.

Ouvrez le fichier openLauncher.html téléchargé.

3.

Puis suivez les informations depuis le launcher.
